# TechJedi LLP Website

3 pages required for Meta App Review submission.

## Files
- `index.html` — Landing page
- `privacy-policy.html` — Privacy Policy (required by Meta)
- `terms.html` — Terms of Service
- `vercel.json` — Clean URL config for Vercel

## Deploy to Vercel (Free, 5 minutes)

1. Go to https://vercel.com → Sign up (free)
2. Click "Add New Project"
3. Upload this folder OR connect GitHub repo
4. Click Deploy → Done!

Your site will be live at: `https://techjedi.vercel.app`
(or connect your own domain: techjedi.in)

## After Deploying

1. Copy your live URL (e.g. https://techjedi.vercel.app)
2. Go to Meta Developer Console → App Settings → Basic
3. Add URL to "App Domains" and "Privacy Policy URL":
   - Privacy Policy URL: https://techjedi.vercel.app/privacy-policy
   - Terms of Service URL: https://techjedi.vercel.app/terms
4. Save → Go to App Review → Submit!

## What to update before deploying
- Replace `contact@techjedi.in` with your real email in all 3 files
- Update the footer address if needed
